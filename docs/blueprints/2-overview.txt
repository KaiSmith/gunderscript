GunderScript Program Design Overview
********************************************************************************

Code Execute Routine:
  - Lex code, convert to array of tokens
    + See the lexer subdirectory for more information.
  - Condense Tokens: produce 
  - Error Checking
  - Compiles code to byte code
  - Executes main function
    + Pushes new frame onto frame_stack
      ~ Allocates frame as well as pointers for all the variables
